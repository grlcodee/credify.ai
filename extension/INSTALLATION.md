# Credify AI Browser Extension - Installation Guide

## üì¶ What You Downloaded

You have downloaded `credify-ai-extension.zip` - the Credify AI Fact Checker browser extension.

## üöÄ Installation Steps

### Step 1: Extract the ZIP File
1. Right-click on `credify-ai-extension.zip`
2. Select **"Extract All..."** or **"Extract Here"**
3. Choose a location on your computer (e.g., Desktop or Documents)
4. You should now have a folder named `extension` with these files inside:
   - `manifest.json`
   - `background.js`
   - `content.js`
   - `popup.html`
   - `popup.js`
   - `popup.css`
   - `icons/` folder

### Step 2: Open Your Browser Extensions Page

**For Google Chrome:**
1. Open Chrome
2. Type in the address bar: `chrome://extensions/`
3. Press Enter

**For Microsoft Edge:**
1. Open Edge
2. Type in the address bar: `edge://extensions/`
3. Press Enter

**For Brave:**
1. Open Brave
2. Type in the address bar: `brave://extensions/`
3. Press Enter

### Step 3: Enable Developer Mode
1. Look for a toggle switch in the **top right corner** that says "Developer mode"
2. Turn it **ON** (it should turn blue/green)

### Step 4: Load the Extension
1. Click the **"Load unpacked"** button (it appears after enabling Developer mode)
2. A file browser window will open
3. Navigate to the folder where you extracted the extension
4. Select the `extension` folder (the one with `manifest.json` inside)
5. Click **"Select Folder"** or **"Open"**

### Step 5: Verify Installation
1. You should see **"Credify AI - Fact Checker"** appear in your extensions list
2. The version should show **1.0.1**
3. Make sure the toggle is turned **ON** (enabled)

## üîÑ Updating the Extension

When you download a new version:
1. Go to your extensions page (`chrome://extensions/`)
2. Find "Credify AI - Fact Checker"
3. Click the **üîÑ Reload** button
4. The new version will be loaded

**Important:** If you see the old design after reloading:
1. Completely **remove** the extension (click "Remove")
2. Close ALL browser windows
3. Reopen your browser
4. Load the unpacked extension again following Steps 2-4 above

## ‚ùì Troubleshooting

### Context menu not appearing?
- Make sure the extension is enabled (toggle is ON)
- Try refreshing the webpage
- Check if "service worker" is active in the extension details

### Old UI still showing?
- The browser may have cached the old files
- Remove the extension completely
- Clear your browser cache (Ctrl + Shift + Delete)
- Reload the extension

### Extension not working?
- Check that you selected the correct folder (should contain `manifest.json`)
- Make sure Developer mode is enabled
- Try disabling and re-enabling the extension
