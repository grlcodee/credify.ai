# **App Name**: Veritas AI

## Core Features:

- Content Analysis: Accept text, links, or content selections to analyze for misinformation.
- Credibility Scoring: Generate a credibility score (0-100) based on AI analysis of the content.
- Fact-Check Verdict: Provide a clear fact-check verdict: True, Misleading, or False, based on AI analysis. The AI will act as a tool and use reasoning to decide when or if to incorporate the result of a search or a call to the fact check API in its analysis
- Verified Summary: Generate a short, verified summary of the content in simple, objective language.  The AI will act as a tool and use reasoning to decide when or if to incorporate information extracted from other web pages or official datasets into the summary
- Evidence Sourcing: Provide links to primary evidence sources (official sources, trusted outlets) used for verification.
- Bias & Emotion Detection: Analyze the content for bias and emotional manipulation techniques.
- Multilingual Support: Offers content analysis and verification in multiple languages.

## Style Guidelines:

- Primary color: Deep blue (#3B5998) to convey trust and authority.
- Background color: Light blue (#E6E9ED), a very desaturated shade of the primary to avoid overpowering the layout.
- Accent color: Violet (#8A2BE2) to highlight key CTAs.
- Body and headline font: 'Inter', sans-serif.
- Use clear, simple icons to represent credibility scores, fact-check verdicts, and evidence sources.
- Clean, intuitive layout with a prominent input field for content analysis.
- Subtle animations to indicate processing and display results.
